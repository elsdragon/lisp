Un marca de ascensores desarrolla un nuevo lenguaje para programarlos. 
El lenguaje se llama "Lots of Insignificant Silly Parenthesis" (LISP) 
e indica a qué planta debe de dirigirse el ascensor, con una cadena que consiste sólo de paréntesis.

Suponiendo que la planta baja es la 0, c
ada vez que se encuentra con un `(` indica que debe de subir una planta, 
y cada vez que se encuentra con una `)` indica que debe de bajar una planta. 
El valor final, es la planta de destino.
